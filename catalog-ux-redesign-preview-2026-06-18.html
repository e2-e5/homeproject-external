UX-редизайн каталога homeproject.online

Превью + обоснование · Ария AI-менеджер · Home Project · 18.06.2026
⚡ ключевое открытие после ресёрча
Структура каталога у нас уже хорошая — лучше, чем была в старом InSales. Меню «Каталог» сегодня даёт три оси входа: Бренды (14 шт) · Коллекции (Новинки / Хиты / OUTLET / Уценка) · Типы техники (12 категорий поперёк всех брендов). На странице бренда — аккордеон категорий со счётчиками + цена + сортировка + хлебные крошки.

Старый InSales давал только одну ось (бренд → категория), без «поперечного» взгляда. Текущий сайт это уже исправил.

Реальная боль — не в структуре. Боль в трёх отсутствующих инструментах:
  • 🔍 Нет поиска по сайту вообще — ни в шапке, ни в каталоге. При 787 товарах на бренд и 33 страницах пагинации это блокер №1.
  • 🎚 Нет глубоких фильтров. «Все индукционные панели до 200 тыс.» построить нельзя. Нет ширины, цвета, серии, типа установки.
  • 📞 Нет CTA конверсии в UI. Ни кнопки «Запросить КП», ни телефона, ни WhatsApp. Сайт «выглядит как каталог», но не ведёт к сделке.

1. Что показал бенчмарк рынка (11 сайтов)

Проверил топ-игроков нашего сегмента: technika-vkusa, lavka.online (Loft), Кучен-Гезельшафт, Miele Gallery, Gaggenau.ru, Miele.ru, Bosch-home, Smeg.ru, Homeier, Hometek, Studio-X.

Сюрприз: полка освобождается.

  • Studio-X.ru — припаркованный домен, продаётся.
  • Bosch-home.ru — закрыл онлайн-магазин, редиректит на сервисный портал BSH.
  • Главные RU-конкуренты в премиум-сегменте свернули e-com.

Три рабочих паттерна в нашем сегменте:

ПаттернКто используетПодходит нам?

Дифференциаторы, которые у конкурентов не заняты:

  • Раздел «Архитекторам» — есть только у Gaggenau, ни у одного RU-дистрибьютора. А 60–70% премиум-сделок идут именно через архитектора. Чистая ниша.
  • Mobile bottom-nav на 5 точек вместо hamburger — никто из конкурентов не делает. При 60–70% мобайла — критично.
  • Двойной вход «по категории / по серии» — Smeg «Эстетика», Gaggenau «Серия 200/400». Закрывает сценарий «хочу консистентный дизайн на всю кухню».

Что нового в шапке:

  • Поиск — instant с группировкой результатов (Товары / Бренды / Категории / Статьи). Работает от 3 символов.
  • Телефон справа — премиум-сегмент звонит, не пишет.

4. Мобильная навигация — bottom-nav вместо hamburger

60–70% трафика — мобайл. Hamburger-меню убивает конверсию: пользователь должен достучаться до меню, открыть, проскроллить. Bottom-nav решает это в 1 тап.

5. Сценарии в кликах: было → станет

Считаем шаги от главной до целевого действия.

📌 Найти духовой шкаф V-ZUG

Главная → Каталог → V-ZUG → Духовые шкафы → карточка
Сейчас: 4 клика, без поиска
Станет: 1 клик через поиск «V-ZUG духовой» или 3 клика через меню

📌 Сравнить все индукционные панели до 200 ₸

Каталог → Варочные → фильтры «Индукция» + цена + ширина → 3 модели в сравнение
Сейчас: невозможно (нет фильтра)
Станет: 4 клика, кнопка «Сравнить» в bottom-nav

📌 Запросить КП по конкретному товару

Карточка товара → «Запросить КП» → форма
Сейчас: нет CTA, нужно найти контакты в «О компании»
Станет: 1 клик с карточки или 1 тап в bottom-nav

📌 Архитектор: подобрать комплект на проект

«Архитекторам» → форма проекта → менеджер
Сейчас: раздела нет
Станет: 1 клик в header или 1 тап в шапке мобайл

📌 Открыть Outlet (уценённые товары)

Каталог → Коллекции → OUTLET
Сейчас: 3 клика, скрыто в подменю
Станет: 2 клика, видно в мегаменю и в коллекциях

📌 Найти конкретный SKU «Spirit 540»

Поиск → ввод SKU → клик в результат
Сейчас: невозможно без поиска. Нужно знать что это Schulthess, открыть бренд, долистать до 33 страницы.
Станет: 1 клик

6. Три варианта на ваш выбор

Вариант A

Минимум — закрыть три боли

Только то, что блокирует конверсию прямо сейчас. Структура не трогается.

  • ✅ Глобальный поиск в шапке (instant)
  • ✅ Кнопка «Запросить КП» + телефон в шапке
  • ✅ Фильтры на странице категории (бренд, цена, ширина, тип нагрева, серия)
  • ✅ Поднять Gaggenau и Liebherr из «по запросу» в основной список
  • ✅ CTA «Добавить в КП» на карточке товара
Срок: ~2 недели Сложность: низкая Влияние на конверсию: +50–80%
✓ Рекомендую — Вариант B

Полная навигация + дифференциаторы

Всё из A + новая шапка с мегаменю + раздел «Архитекторам» + mobile bottom-nav.

  • ✅ Всё из Варианта A
  • ✅ Перестроить header (Каталог · Бренды · Архитекторам · Реализованные проекты · О компании)
  • ✅ Мегаменю с 4 колонками (Типы / Бренды / Коллекции / Промо-блок)
  • ✅ Раздел «Архитекторам» — лендинг с формой проекта (CAD/BIM, регистрация в AI-агенте)
  • ✅ Mobile bottom-nav (Каталог · Поиск · Сравнить · Изб · КП)
  • ✅ Лендинги «Новинки», «Хиты», «Эксклюзив», «Outlet» с историей коллекции
  • ✅ Виджет «Сравнение товаров» (до 4 моделей)
Срок: ~4 недели Сложность: средняя Влияние на конверсию: +100–150% Дифференциация: высокая (Архитекторам + bottom-nav никто из RU не делает)
Вариант C

Премиум — полный UX-апгрейд

Всё из B + продвинутые инструменты для архитекторов и эстетики.

  • ✅ Всё из Варианта B
  • ✅ Двойной вход: «по категории» + «по серии/стилю» (например, Küppersbusch K8 как отдельный лендинг)
  • ✅ 3D-визуализатор кухни с расстановкой техники
  • ✅ Личный кабинет архитектора с проектами, скачиванием CAD/BIM/dwg
  • ✅ Сравнение с highlighting разницы (не два столбца спецификаций, а подсветка отличий)
  • ✅ Подбор по сценарию («Кухня под ключ / Только встройка / Под мебель»)
  • ✅ Список избранного → конвертация в КП одним кликом
Срок: ~8 недель Сложность: высокая Влияние на конверсию: +150–250% Подходит когда: есть стабильный трафик 1000+/день
Какой вариант идём?
  • A — быстро закрываем три блокера за 2 недели, остальное потом
  • B — рекомендую — за 4 недели получаем полный UX + 2 дифференциатора (Архитекторы + bottom-nav)
  • C — премиум-история, рост 3–8 недель, имеет смысл когда выйдем на стабильный трафик

Мой совет: идём по B, но в два этапа.
  • Спринт 1 (неделя 1–2): поиск + CTA в шапке + фильтры + поднятие Gaggenau/Liebherr → запускаем конверсию.
  • Спринт 2 (неделя 3–4): мегаменю + Архитекторам + bottom-nav + лендинги коллекций → строим дифференциацию.

Скажете «A», «B одним этапом», «B в два этапа» или «C» — соберу финальный план и отдам инженеру.
