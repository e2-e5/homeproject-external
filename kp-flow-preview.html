Шаг 4 — proposal-writer агент генерирует КП
1

~1 сек
2

Поиск SKU в Medusa по категориям и бренду

GET /store/products?brand=Küppersbusch&q=духовой → топ-3 SKU → variants[].items[]

~3–5 сек
3

Генерация текста (LLM)

Быстрое — 2 поля: introTitle + introText. Полное — ~25 полей: brief, варианты A/B, визуальные карточки, рекомендации

Быстрое: ~15 сек Полное: ~80 сек
4

Рендеринг HTML + отправка

kp-render-simple.ts или kp-render-full.ts → HTML → email клиенту + уведомление менеджеру

~2 сек
5

Обновление статуса

kp_requests.status = 'sent', kpId = id, kpUrl = ссылка, sentAt = now()

~1 сек
