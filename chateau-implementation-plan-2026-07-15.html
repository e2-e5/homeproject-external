🌐 Часть 2 — Интерактивный / «игровой» сайт
🔍 Исходная ситуация (из анкеты)

У Château уже есть интернет-магазин с клубом привилегий и накопительными картами. Есть кулинарный салон с записью через соцсети. Проблемы цифрового канала:

  • Сайт не генерирует квалифицированных лидов — клиенты приходят walk-in или по рекомендации
  • МК не монетизируются через сайт — запись через Instagram/Telegram, контакты не собираются
  • Техника не продаётся онлайн — нет инструмента сравнения, подбора, конфигуратора
  • Онлайн-чат — живой оператор (указали в анкете) — нет AI-ассистента
